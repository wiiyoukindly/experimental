# experimental

Experimental workspace for plans, ideas, and exploratory work co-authored with Claude Code.

## Structure

- `plans/` — planning documents from Claude Code sessions. Naming convention: `YYYY-MM-DD_<slug>.md`.

## How plans land here

Plans are drafted in `C:/Users/LAPTOP/.claude/plans/` during a Claude Code session and committed here at the end of the session with a datestamped filename.
